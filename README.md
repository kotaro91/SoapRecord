# 固形石鹸（ボディソープ向け）50選 比較調査プロジェクト

本プロジェクトは、高品質なナチュラルクラフトソープ専門店である「THE COLD PROCESS」（1個あたり3,300円〜4,400円程度）のような、**肌に優しく香りが良い**という特徴を持ちつつ、**少し価格帯を落とした（400円〜2,500円程度）**代替となる固形石鹸50種類を調査・整理したものです。

## THE COLD PROCESS と本調査の狙い
「THE COLD PROCESS」の石鹸は、熱を加えず低温でじっくり熟成させる**コールドプロセス製法**により、植物オイルの有用成分や天然の保湿成分（グリセリン）を壊さずに閉じ込め、豊かなアロマ精油の香りが特徴です。
本調査では、同じくコールドプロセス製法やそれに準ずる良質な製法（枠練りなど）を採用し、天然の香りと低刺激性にこだわった、より日常使いしやすい価格帯の石鹸を厳選しました。

## Google スプレッドシート ＆ GAS によるデータ同期（データ消失防止）
ブラウザのキャッシュやCookieを削除しても、お気に入りや購入記録が消えないようにするため、Google スプレッドシートとクラウド経由で同期する機能を搭載しています。

### 1. スプレッドシートの準備
1. [Google スプレッドシート](https://sheets.google.com/)を新規作成します。シート名は「シート1」のままで構いません。
2. 作成したスプレッドシートのURLをブラウザのメモなどにコピーしておきます。

### 2. Google Apps Script (GAS) のセットアップ
1. スプレッドシート上部メニューの **「拡張機能」 ＞ 「Apps Script」** をクリックします。
2. もともと入力されている `function myFunction() {}` などのコードをすべて消去します。
3. 以下の **「GASコピペ用ソースコード」** をすべてコピーし、エディタに貼り付けます。
4. **【重要】** コードの1行目にある `var SECURITY_PASSWORD = "ここにパスワードを入力";` の部分を、ご夫婦で決めた**「共通の秘密のパスワード」**に書き換えてください。
5. プロジェクトの保存（エディタ上のフロッピーディスクのアイコン）をクリックします。

#### 【GASコピペ用ソースコード】
```javascript
// 【必須設定】ご夫婦で共有する秘密のパスワードを設定してください（例: "mySecretSoapKey123"）
var SECURITY_PASSWORD = "ここにパスワードを入力";

function doGet(e) {
  var password = e.parameter.password;
  if (password !== SECURITY_PASSWORD) {
    return ContentService.createTextOutput(JSON.stringify({ status: "error", message: "Unauthorized: Invalid password" }))
      .setMimeType(ContentService.MimeType.JSON);
  }

  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = sheet.getDataRange().getValues();
  var result = {};
  
  // 1行目はヘッダー [ID, お気に入り, 購入済み, 購入日, 感想メモ, 使用開始日, 使用終了日, 購入金額]
  // 2行目以降がデータ
  for (var i = 1; i < data.length; i++) {
    var id = data[i][0];
    if (!id) continue;
    result[id] = {
      favorite: data[i][1] === true || data[i][1] === "TRUE" || data[i][1] === 1,
      purchased: data[i][2] === true || data[i][2] === "TRUE" || data[i][2] === 1,
      date: data[i][3] ? formatDate(data[i][3]) : "",
      notes: data[i][4] || "",
      startDate: data[i][5] ? formatDate(data[i][5]) : "",
      endDate: data[i][6] ? formatDate(data[i][6]) : "",
      price: data[i][7] !== undefined && data[i][7] !== "" ? Number(data[i][7]) : ""
    };
  }
  
  return ContentService.createTextOutput(JSON.stringify(result))
    .setMimeType(ContentService.MimeType.JSON);
}

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  
  // スプレッドシートが空、またはヘッダーしかない場合は初期化
  var lastRow = sheet.getLastRow();
  if (lastRow <= 1) {
    sheet.getRange(1, 1, 1, 8).setValues([["ID", "お気に入り", "購入済み", "購入日", "感想メモ", "使用開始日", "使用終了日", "購入金額"]]);
    // 50行分を初期作成
    var initialValues = [];
    for (var id = 1; id <= 50; id++) {
      initialValues.push([id, false, false, "", "", "", "", ""]);
    }
    sheet.getRange(2, 1, 50, 8).setValues(initialValues);
  }
  
  var postData = JSON.parse(e.postData.contents);
  var password = postData.password;
  if (password !== SECURITY_PASSWORD) {
    return ContentService.createTextOutput(JSON.stringify({ status: "error", message: "Unauthorized: Invalid password" }))
      .setMimeType(ContentService.MimeType.JSON);
  }
  
  // データをシートから読み込み、IDと行番号のマップを作る
  var dataRange = sheet.getDataRange();
  var values = dataRange.getValues();
  var idToRow = {};
  for (var i = 1; i < values.length; i++) {
    idToRow[values[i][0]] = i + 1; // 1-based index for row
  }
  
  // 送られてきたデータを書き込む
  var favorites = postData.favorites || [];
  var purchased = postData.purchased || {};
  
  // 1〜50のIDについて更新用の配列を作成
  for (var id = 1; id <= 50; id++) {
    var isFav = favorites.indexOf(id) !== -1 || favorites.indexOf(String(id)) !== -1;
    var pInfo = purchased[id] || purchased[String(id)];
    var isBought = !!pInfo;
    var date = pInfo ? pInfo.date : "";
    var notes = pInfo ? pInfo.notes : "";
    var startDate = pInfo ? pInfo.startDate : "";
    var endDate = pInfo ? pInfo.endDate : "";
    var price = pInfo && pInfo.price !== undefined && pInfo.price !== "" ? pInfo.price : "";
    
    // スプレッドシートの行を特定して更新 (F, G, H列を含むように範囲を拡大: B列からH列の計7列分)
    var row = idToRow[id];
    if (row) {
      sheet.getRange(row, 2, 1, 7).setValues([[isFav, isBought, date, notes, startDate, endDate, price]]);
    } else {
      // 行がなければ追加
      sheet.appendRow([id, isFav, isBought, date, notes, startDate, endDate, price]);
    }
  }
  
  return ContentService.createTextOutput(JSON.stringify({ status: "success" }))
    .setMimeType(ContentService.MimeType.JSON);
}

function formatDate(dateObj) {
  if (dateObj instanceof Date) {
    var year = dateObj.getFullYear();
    var month = ("0" + (dateObj.getMonth() + 1)).slice(-2);
    var day = ("0" + dateObj.getDate()).slice(-2);
    return year + "-" + month + "-" + day;
  }
  return String(dateObj);
}
```

### 3. ウェブアプリとしてのデプロイ
1. Apps Script 画面 of 右上にある **「デプロイ」 ＞ 「新しいデプロイ」** をクリックします。
2. 左上の歯車アイコン（種類の選択）をクリックし、 **「ウェブアプリ」** を選択します。
3. 設定項目を以下のように指定します：
   - 説明: `Soap Finder Sync` (任意)
   - 次のユーザーとして実行: **「自分」**
   - アクセスできるユーザー: **「全員」** (※ブラウザからの直接アクセスを許可するために「全員」にする必要があります)
4. **「デプロイ」** をクリックします。
5. 初回は「アクセスを承認」する画面が表示されます。
   - 「アクセスを承認」を押し、自分のGoogleアカウントを選択します。
   - 「このアプリは Google で確認されていません」と表示された場合は、左下の **「詳細」 ＞ 「（安全ではないページ）に移動」** をクリックし、アクセス権限を **「許可」** します。
6. デプロイ完了画面に表示される **「ウェブアプリのURL」** (最後に `/exec` で終わるURL) をコピーします。

### 4. ダッシュボードでの設定
1. ブラウザで [dashboard.html](file:///Users/kk/Documents/MyVault/Project/SoapProject/dashboard.html) を開きます。
2. 画面上部の **「☁️ スプレッドシート同期設定」** アコーディオンをクリックして開きます。
3. コピーしたウェブアプリURLを入力欄に貼り付けます。
4. 手順2-4で設定した **「同期用パスワード（合言葉）」** も入力欄に貼り付けます。
5. **「設定を保存して同期」** をクリックします。
6. 「Googleスプレッドシートとの同期設定が成功しました！」と表示されれば完了です。

## 🌐 スマホ・iPadからアクセスする方法（GitHub Pagesによる公開）
Safariのお気に入り（ブックマーク）に追加して簡単に開くには、GitHub Pages（無料ホスティング）を利用するのが最も簡潔です。

### 1. GitHubリポジトリへのアップロード
GitHubのアカウントをお持ちであれば、以下のGitコマンドでこのプロジェクトをアップロードできます：
```bash
# プロジェクトディレクトリに移動
cd /Users/kk/Documents/MyVault/Project/SoapProject

# Gitの初期化
git init
git add .
git commit -m "Initial commit for SoapProject"

# GitHubで新規リポジトリを作成し、以下を実行
git branch -M main
git remote add origin https://github.com/[あなたのGitHubアカウント名]/[リポジトリ名].git
git push -u origin main
```

### 2. GitHub Pagesの設定
1. GitHubの該当リポジトリのページを開き、上部の **「Settings」** タブをクリックします。
2. 左メニュー of **「Pages」** をクリックします。
3. **Build and deployment** セクションの **Branch** を「None」から **「main」** に変更し、その右のフォルダを「/ (root)」のまま **「Save」** ボタンを押します。
4. 数分待つと、画面上部に公開URL（例: `https://[アカウント名].github.io/[リポジトリ名]/dashboard.html`）が生成されます。

### 3. 各端末でのブックマーク
1. 生成されたURLをiPhoneやiPad、PCのSafariで開きます。
2. Safariの下部共有アイコンから **「お気に入りに追加」** または **「ホーム画面に追加」** を選択します。
3. クラウド同期パネルに、設定した「GASのURL」と「パスワード」を入力して同期すれば、すべての端末から同じ情報に簡単にアクセス・共同編集ができます。
